## Optional Downloaded Components

AspectShift-HtoV includes optional subtitle generation features that may require additional third-party components to be downloaded after installation.

These components are **not bundled with the standard installer** and are only downloaded when subtitle-related functionality is explicitly requested by the user.

### whisper.cpp

Project:

https://github.com/ggerganov/whisper.cpp

whisper.cpp provides local speech-to-text functionality used for subtitle generation.

### OpenAI Whisper Models

Project:

https://github.com/openai/whisper

AspectShift-HtoV may download compatible Whisper model files, including GGML-based models, to support offline subtitle generation and export features.

Copyright for whisper.cpp, Whisper, and associated model files remains with their respective authors, contributors, and copyright holders.

Users should consult the respective upstream projects for complete licensing information, source code availability, and applicable usage terms.
